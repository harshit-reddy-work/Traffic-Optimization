# ⚡ Quick Start - Traffic Control App

## 🚀 3 Simple Steps:

### 1. Run the Application
```powershell
python traffic_control_app.py
```

### 2. Upload Videos
- Click "📁 Upload Video" for each lane
- Select your videos (e.g., from `videos/` folder)

### 3. Process & Calculate
- Click "🚀 Process All Videos"
- Wait for completion
- Click "⚡ Calculate Timing"
- Watch the simulation! 🎬

---

## ✨ That's It!

The application will:
- ✓ Detect vehicles in your videos
- ✓ Count them per lane
- ✓ Calculate optimal light timing
- ✓ Show live traffic light switching

**Ready to optimize your traffic flow!** 🚦
