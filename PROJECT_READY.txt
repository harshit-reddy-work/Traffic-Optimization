================================================================================
🎯 WHAT IT DOES
================================================================================

1. Opens video file
2. Detects vehicles (car, bus, truck, bike, motorcycle)
3. Tracks them across frames
4. Assigns unique IDs to each vehicle
5. Counts total vehicles
6. Displays results in real-time

================================================================================
🎮 CONTROLS
================================================================================

Press Q - Quit/Exit program
Press P - Pause video

================================================================================
🚀 TO RUN MULTIPLE VIDEOS
================================================================================

Edit multithreading_yolo8.py line 120:
  total = countVehicles("/videos/your_video.mp4")

Save and run again.

To process 4 lanes (like original):
  - Run 4 times with different video paths
  - Collect the output counts
  - Use program.py to calculate timing
